## How to View the Flowcharts

### Option 1: Markdown Viewers (Recommended)
These flowcharts use Mermaid syntax, which is supported by:

**Online Mermaid Live Editor:**
- Visit: https://mermaid.live
- Copy and paste the Mermaid code blocks to see interactive diagrams

**GitHub/GitLab:**
- Open the `.md` files directly in your repository
- Mermaid diagrams render automatically in GitHub/GitLab

### Option 2: VS Code Extensions
Install these extensions in VS Code:
- **Mermaid Markdown Syntax Highlighting**
- **Mermaid Editor**
- **Markdown Preview Enhanced**

### Option 3: Documentation Platforms
- **Notion**: Supports Mermaid diagrams in markdown blocks
- **Confluence**: With Mermaid plugin
- **Markdown editors**: Typora, Mark Text, etc.

### Option 4: Browser Extensions
- **Mermaid Live Editor** browser bookmark
- **Markdown preview extensions** with Mermaid support
